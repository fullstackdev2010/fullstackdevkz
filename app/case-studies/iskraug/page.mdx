# Iskra UG – Mobile Trading Suite

A modern **cross-platform trading application** with a **FastAPI backend** and Expo-based mobile client.  
It provides real-time catalog browsing, cart management, and offline order processing.

## Overview
Iskra enables industrial clients and field agents to browse multi-level product catalogs, manage carts, and track orders seamlessly.  
It is powered by a **FastAPI + PostgreSQL** backend and a **React Native / Expo** frontend, integrated via REST APIs.

## Key Features
- Hierarchical catalog and full-text search  
- Cart persistence and merge on login  
- Order creation, confirmation, and tracking  
- Offline cache with background revalidation  
- Secure JWT authentication with refresh tokens  
- EAS build pipelines for deployment  

## Technical Highlights
- **Frontend:** Expo 53 · React Native 0.79 · TypeScript  
- **State:** Zustand + TanStack Query  
- **Styling:** Tailwind (nativewind) + Paper UI  
- **Backend:** FastAPI · SQLAlchemy · Pydantic · PostgreSQL  
- **Middleware:** CORS, rate-limiting (slowapi), async routers  
- **DevOps:** EAS pipelines · automated seeds · CI-ready scripts  

## Result
A production-grade mobile + backend ecosystem for secure trading.  
Iskra UG ensures resilient offline access, fast synchronization, and complete data integrity for enterprise-level deployments.
