# Iskra (Mobile)

A production‑grade **Expo** application for catalog browsing, orders and account management with **real‑time state** and **offline tolerance**.

## Highlights
- **Catalog** with hierarchical navigation & search
- **Orders** workflow with cart and checkout
- **Resilient networking**: axios + retry, online/offline detection
- **State** with **Zustand**, data queries via **TanStack Query**

## Stack
- **Expo 53**, React Native 0.79, TypeScript
- Navigation: **@react-navigation/native**, bottom tabs
- Styling: **Tailwind (nativewind)**, Paper UI
- Data: **axios**, **@tanstack/react-query**, **zustand**
- Device: NetInfo, SecureStore, Haptics, Reanimated/Gesture

## What we delivered
- Smooth UX with cached data & graceful retries
- Secure auth and session handling
- Production build pipeline with EAS profiles
