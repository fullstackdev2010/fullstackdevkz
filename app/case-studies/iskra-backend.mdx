# Iskra (Backend)

A clean, modular **FastAPI** backend with **SQLAlchemy** models, authentication, and rate limiting — powering the Iskra mobile client.

## Highlights
- **Auth** with refresh tokens
- **Domain models**: users, trade items, category codes, orders, pictures
- **Routers**: `/auth`, `/orders`, `/trade`, `/category-code`, `/pictures`
- **CORS** + **rate limiting** (slowapi), clean health checks

## Stack
- **FastAPI**, **Pydantic**, **SQLAlchemy**
- ASGI server (Uvicorn), CORS middleware
- PostgreSQL via SQLAlchemy engine
- Scripts for data loaders & DB init

## What we delivered
- Predictable, documented REST API surface
- Secure session model with refresh tokens
- DB migrations & seed scripts for rapid bootstrap
